# Changelog

## 1.0.0 (2024)

- Initial release of QuickDoc Preview
- Modern PDF viewer with VSCode theme integration
- Custom toolbar buttons: Reset Zoom (R), Fit to Width (F), Open in Browser
- Clean, organized project structure
- Hidden scrollbars for distraction-free viewing
- Seamless dark/light theme adaptation

## Previous Versions

This extension is based on the original vscode-pdf extension with significant improvements and modernization.
